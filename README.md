# Dirt Money

Dirt Money is a standalone browser farming-business RPG prototype set in Ash Creek County.

## Run

```bash
npm run dev
```

Then open `http://localhost:5173`.

The game has no external runtime dependencies.

## Test

```bash
npm test
```

## Current Scope

- Title screen, menu, new game, background selection, settings
- Farm dashboard with weekly advancement and reports
- Multiple fields with planting, scouting, soil tests, fertility, weeds, stress, growth, harvest, and stored crops
- Harvest-on-credit behavior when cash is insufficient
- Grain elevator crop sales
- Cash, debt, operating credit, debt payment, and credit draw
- Salvage purchase plus sell, strip, repair/flip, and use-on-equipment exits
- Equipment condition and repairs with cash, credit, or parts
- Contracts with deadlines, requirements, rewards, failure consequences
- Ash Creek County locations and core NPC interactions
- Local save/load through browser storage
- Dirt Money visual palette and placeholder assets

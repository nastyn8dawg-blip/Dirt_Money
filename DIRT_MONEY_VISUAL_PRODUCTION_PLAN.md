# Dirt Money Visual Production Plan

This file records the active visual plan used for implementation because the referenced external plan file was not present in the workspace.

## Direction

Dirt Money should look like a weathered farm ledger and rural CRPG interface: practical, grounded, warm, and worn. The game should avoid cute farming-game visuals, glossy mobile styling, fantasy decoration, neon color, and modern SaaS dashboard language.

## Palette

- Coffee Black: `#15110C`
- Burnt Coffee: `#241A12`
- Dark Walnut: `#332418`
- Weathered Leather: `#4A3321`
- Warm Cream: `#E8D8B8`
- Dust Gray: `#9A907F`
- Harvest Gold: `#D6A73A`
- Barn Red: `#A64A35`
- County Green: `#7FA36A`
- Steel Blue: `#6F8794`
- Aged Brass: `#8A6A35`
- Ash Gray: `#5D5850`

## UI Rules

- Use dark coffee panels and aged brass borders.
- Use cream text for primary readability.
- Use harvest gold for important productive actions.
- Use barn red for warnings and financed-risk choices.
- Use county green for positive reputation and completed work.
- Use steel blue for weather, market, and informational notes.
- Keep panels dense, legible, and practical.
- Do not bake functional text into images.
- Keep placeholder assets clearly named and replaceable.

## Placeholder Asset Naming

- UI panel placeholders: `assets/placeholders/ui-panel-*`
- Functional icons: `assets/placeholders/icons/{function}.svg`
- Final art prompts: `assets/source/prompts/final-art-prompts.md`
